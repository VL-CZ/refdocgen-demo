public void Emit(LogEvent logEvent) #
Implements ILogEventSink.Emit.Emit
Emit the provided log event to the sink. If the sink is being disposed or the app domain unloaded, then the event is ignored.

Parameters

LogEvent logEvent
Log event to emit.
Remarks
The sink implements the contract that any events whose Emit() method has completed at the time of sink disposal will be flushed (or attempted to, depending on app domain state).
