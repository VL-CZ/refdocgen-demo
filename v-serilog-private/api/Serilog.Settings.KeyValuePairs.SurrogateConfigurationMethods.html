internal static class SurrogateConfigurationMethods

assembly Serilog
namespace Serilog.Settings.KeyValuePairs
Base type:  object
Contains "fake extension" methods for the Serilog configuration API. By default the settings knows how to find extension methods, but some configuration are actually "regular" method calls and would not be found otherwise. This static class contains internal methods that can be used instead. See also CallableConfigurationMethodFinder
