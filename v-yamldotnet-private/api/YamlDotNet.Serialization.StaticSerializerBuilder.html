public sealed class StaticSerializerBuilder

assembly YamlDotNet
namespace YamlDotNet.Serialization
Base type:  StaticBuilderSkeleton<StaticSerializerBuilder>
Creates and configures instances of Serializer. This class is used to customize the behavior of Serializer. Use the relevant methods to apply customizations, then call StaticSerializerBuilder.Build to create an instance of the serializer with the desired customizations.
